▶ 사용기술 <br>
- **Language:** Swift, php
- **UI:** UIKit, Autolayout
- **Libary:** Alamofire, BSImagePicker, Kingfisher, swiftyJSON, NaverMapAPI
- **Server :** AWS / Amazon EC2 - OS: Ubuntu 18.04.5
- **Database :** Mysql
- **Protocol :** HTTP  

▶ 주요기능<br> 
- 스플래시 화면<br>
- 소개페이지<br>
- 로그인, 로그아웃, 회원가입, 아이디 중복확인, 탈퇴<br>
- 게시글 작성, 다중이미지 업로드, 수정, 삭제 , 현재위치표기 , 검색,  신고, 페이징<br> 
- 댓글 작성, 삭제 , 신고<br> 
- 게시글 북마크 기능<br> 
- 내가 쓴글 모아보기<br>
